#  Income & Expense Management System
A backend-focused application developed as part of a company assignment to manage personal income and expenses efficiently. This project demonstrates core backend concepts including REST API design, authentication, and database operations.
##  Features
- Add, update, and delete income/expense transactions  
- Categorize transactions (e.g., Food, Salary, Bills)  
- Track total income, expenses, and balance  
- Filter transactions by date and category  
- Secure user authentication  
##  Architecture
- Layered Architecture:
  - Controller → Service → Repository  
- Clean separation of concerns  
- Scalable and maintainable design  
##  Tech Stack
- **Backend:** Spring Boot  
- **Persistence:** Spring Data JPA, Hibernate  
- **Database:** MySQL  
- **Security:** Spring Security, JWT  
- **Tools:** Maven, Postman, Git  
##  Security
- JWT-based authentication  
- Role-based access control (RBAC)  
- Secure API endpoints 
##  API Capabilities
- RESTful API design  
- CRUD operations for transactions  
- Filtering support  
- Structured JSON responses
